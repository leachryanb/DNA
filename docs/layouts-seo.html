Page Layouts Optimized for SEO

The following layouts maintain the Content A, Content B, Content C document order but change the visual order. This is a benefit for screen reader users as well as increasing SEO by positioned the main content higher in the document order. With the main content closer to the top of the page, web crawlers and screen read users do not have to weed through additional (and usually repetative) content such as navigation links or other sidebar items before getting to the meat of the page.
